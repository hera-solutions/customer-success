# Hera Logo Assets

Brand-approved Hera logo files for use in PDF reports, email signatures, slide decks, knowledge base articles, and other Hera-branded deliverables. Pulled from the Marketing team's "Logos LATEST / Black Circle Removed" set.

For brand colors and typography, see [`branding/brand-guidelines.md`](../brand-guidelines.md).

All files have transparent backgrounds.

## When to use which file

| File | Dimensions | Best for |
|---|---|---|
| `Hera Logo - Horizontal-1.png` | 640 x 244 | Default full-brand header. Includes the dog mark, "hera" wordmark, and the "a dsp's best friend" tagline. Use on PDF cover pages, slide title slides, and any document where space allows the full lockup. |
| `Hera Logo - Horizontal - No Tag Line-1.png` | 640 x 234 | Compact horizontal lockup without the tagline. Use for in-document headers, repeating page headers, and tight horizontal spaces where the tagline would clutter the layout. |
| `Hera Logo - Vertical-1.png` | 640 x 593 | Square-ish vertical lockup with tagline. Use for cover pages, square social posts, certificates, and centered layouts. |
| `Hera Mark - Dog Circle.png` | 219 x 233 | Dog-only avatar (the pink circle mark). Use for profile pictures, favicons, app icons, signature thumbnails, and bullet-point spot graphics. |
| `Hera Wordmark.png` | 420 x 228 | "hera" text only, no dog. Use when the dog mark already appears elsewhere on the page or when the brand name needs to read at a glance without the illustration. |
| `Hera Sketch-1.png` | 305 x 640 | Full-body Hera illustration with no circle and no text. Use as a decorative spot illustration, in margins, or as a friendly accent on internal docs. |
| `Hera is here to help-1.png` | 640 x 280 | "hera is here to help" marketing graphic. Use for support announcements, knowledge base banners, and customer-facing campaigns. |
| `Hera is here to help-SMALLER.png` | 320 x 140 | Half-resolution version of the "here to help" graphic. Use for email signatures and small inline placements where the full-size file is overkill. |

## Default picks by deliverable

- **PDF cover page:** `Hera Logo - Vertical-1.png` (centered) or `Hera Logo - Horizontal-1.png` (top-aligned).
- **PDF page header (repeating):** `Hera Logo - Horizontal - No Tag Line-1.png`.
- **Email signature:** `Hera is here to help-SMALLER.png`.
- **Knowledge base article banner:** `Hera is here to help-1.png`.
- **Profile picture / avatar:** `Hera Mark - Dog Circle.png`.
- **Slide deck title slide:** `Hera Logo - Horizontal-1.png` or `Hera Logo - Vertical-1.png` depending on layout.

## Notes

- These are rasterized PNG exports. If you need vector (SVG, AI, EPS) versions for resizing without quality loss, ask Marketing.
- The "Black Circle Removed" set is the version where the dog sits on a transparent background instead of a solid black circle. If a different background treatment is needed, check the Marketing folder for alternative sets before exporting your own.
